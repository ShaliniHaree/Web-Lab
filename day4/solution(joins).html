SQL Joins

Question 1

Which type of join returns all rows from both tables, joining records where available and leaving NULL values where not available?

Answer : FULL OUTER JOIN

Question 2

In SQL, what is the purpose of the JOIN keyword?

Answer : To combine columns from multiple tables

Question 3

Which join returns all rows from the left table and matching rows from the right table?

Answer :LEFT JOIN

Question 4

What happens if you perform a CROSS JOIN between two tables?

Answer : It returns the Cartesian product of the two tables
