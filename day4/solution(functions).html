Functions and Recursion

Question 1

What is the output of the following C code snippet?

#include
int foo(int n) {
if (n <= 0)
return 0;
else
return n + foo(n - 1);
}

int main() {
printf("%d\n", foo(4));
return 0;
}



Answer : 24

Question 2

What is the purpose of the following C function?

int factorial(int n) {
if (n <= 1)
return 1;
else
return n * factorial(n - 1);
}



Answer : To calculate the factorial of a number.
