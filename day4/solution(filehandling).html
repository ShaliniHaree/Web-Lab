Which of the following functions is used to open a file in C for writing?

Answer : fopen("file.txt", "w")

Question 2

What does the fseek() function do in C file handling?

Answer : Moves the file pointer to a specified location

Question 3

Which of the following functions is used to close a file in C?

Answer : fclose()

Question 4

What does the feof() function in C file handling indicate?

Answer : End of file has been reached
