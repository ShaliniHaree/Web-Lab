Pointers

Question 1

In C, how do you dereference a pointer variable?

Answer : By using the asterisk (*) symbol before the pointer variable

Question 2

Which of the following is the correct syntax to access the value stored in the variable pointed to by a pointer in C?

Answer : *ptr

Question 3

What is the output of the following C code?
#include
int main() {
int x = 5;
int *ptr = &x;
printf("%d\n", *ptr);
return 0;
}

Answer :5

Question 4

What is the purpose of the sizeof operator when used with pointers in C?

Answer : Returns the size of the pointer variable itself
