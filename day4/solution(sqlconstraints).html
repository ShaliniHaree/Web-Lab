SQL Constraints

Question 1

What does the SQL constraint PRIMARY KEY enforce?

Answer : Ensures uniqueness of values in a column

Question 2

Which constraint ensures that a column cannot have NULL values?

Answer : NOT NULL

Question 3

What does the EXISTS operator do in SQL?

Answer :Ensures that values in a column are unique

Question 4

What does the SQL CHECK constraint do?

Answer : Limits the range of values that can be inserted into a column
