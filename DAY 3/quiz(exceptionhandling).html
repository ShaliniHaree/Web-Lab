Exception Handling

Question 1

What is an exception in Python?

  • A logical error in the code
  • A syntax error in the code
  • An unexpected event that occurs during program execution
  • A comment in the code

Question 2

How do you handle exceptions in Python?

  • Using the catch keyword
  • Using the try and except blocks
  • Using the finally block
  • Using the raise keyword

Question 3

What does the finally block in exception handling do?

  • Executes the code inside the block if an exception occurs
  • Executes the code inside the block regardless of whether an exception occurs
  • Raises an exception
  • Terminates the program

Question 4

What is the purpose of the else block in exception handling?

  • Executes the code inside the block if no exceptions occur
  • Executes the code inside the block if an exception occurs
  • Raises an exception
  • Terminates the program
