Exception Handling

Question 1

What is an exception in Python?

Answer : An unexpected event that occurs during program execution

Question 2

How do you handle exceptions in Python?

Answer : Using the try and except blocks

Question 3

What does the finally block in exception handling do?

Answer : Executes the code inside the block regardless of whether an exception occurs

Question 4

What is the purpose of the else block in exception handling?

Answer : Executes the code inside the block if no exceptions occur
