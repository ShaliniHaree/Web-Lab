Expressions

Question 1

What is a regular expression in Python?

Answer : A sequence of characters that define a search pattern

Question 2

What module in Python is used for working with regular expressions?

Answer : 're'

Question 3

What function is used to search for a pattern in a string using regular expressions?

Answer : search()

Question 4

What does the re.sub() function do in Python regular expressions?

Answer : Substitutes a pattern in a string with another string
