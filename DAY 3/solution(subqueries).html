SQL Subqueries

Question 1

What is a subquery in SQL?

Answer : A query that is nested inside another query

Question 2

Which operator is used to compare a value to a list of values returned by a subquery?

Answer : IN

Question 3

What does the finally block in exception handling do?

Answer :Checks if a subquery returns any rows

Question 4

Which keyword is used to alias a subquery in SQL?

Answer : AS
